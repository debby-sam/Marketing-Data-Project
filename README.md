Project Overview
This project integrates advertising data from Facebook, Google, and TikTok into a unified dataset and visualizes key performance metrics in a Power BI one-page dashboard.
The dashboard provides a consolidated view of impressions, clicks, spend, conversions, views, and reach, enabling cross-channel performance analysis and actionable insights.

Features
Data Integration: Unified table combining multiple ad platforms.

Date Dimension: Custom Date Table for time-based analysis.

KPIs: Impressions, Clicks, Spend, Views, Conversions, CTR, CPM, CPC, CPA.

Visuals:

KPI cards for headline metrics

Bar charts for spend and impressions by campaign/ad set

Line charts for daily trends (impressions, clicks, CPM)

Donut chart for conversions by campaign

Table for detailed campaign/ad set breakdown

Slicers: Interactive filters for Platform, Campaign, Ad Set, and Date.

Insights
Weekday dominance: Tuesdays and Mondays deliver peak impressions and clicks.

Budget allocation: Influencer and conversion-focused ad sets consume the largest share of spend.

Campaign performance: Influencer_CoL and Awareness_GenZ campaigns drive the highest impressions.

Audience reach: Broad Audience (18–35) and Video Enthusiasts segments deliver the widest reach.

Recommendations
Reallocate budget toward high-performing campaigns and reduce spend on underperformers.

Increase ad delivery on Tuesdays and Mondays; reduce weekend spend.

Benchmark CPM, CPC, and CPA across platforms to identify cost-efficient channels.

Continue investing in broad audiences and video enthusiasts while testing niche segments carefully.

Optimize conversions with creative testing and retargeting strategies.

Getting Started
Prerequisites
Power BI Desktop (or Power BI Service)

CSV files containing Facebook, Google, and TikTok ad data

Steps
Import the CSV files into Power BI.

Create an Append marketing table to unify the datasets.

Build a Date Table using CALENDARAUTO() or a custom range.

Define DAX measures for KPIs (e.g., Total Impressions, CPM, CTR).

Design the dashboard using the one-page layout described above.

Add slicers for interactivity.

Publish to Power BI Service and record a walkthrough video.

Deliverables
Dashboard Link: Published Power BI dashboard (shared via Power BI Service).
